# Basic-Front-end-web-development-projects
The collections of HTML+CSS+JS projects. In this project there are Web Calculator, Clone Login Page of Facebook, Web to-do list app, Web Stop watch app, Shopping app and Web music player with default songs. These programs are my very first step into programming. Especially in web dev. These are made by watching tutorial videos of 2 legend YouTubers.
